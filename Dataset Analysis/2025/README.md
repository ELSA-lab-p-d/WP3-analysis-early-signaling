# Early-Warning Outreach Analysis

This notebook presents an exploratory and descriptive analysis of municipal early-warning outreach data related to debt interventions. The analysis focuses on key outreach outcomes, including successful contact and help acceptance.

Using summary statistics, grouped comparisons, and simple statistical models, the notebook examines patterns in outreach performance across time, municipalities, and selected individual characteristics. The workflow is designed to be transparent and reproducible, supporting monitoring and further evaluation of early-warning outreach practices.
